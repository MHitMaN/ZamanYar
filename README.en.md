## Overview
ZamanYar is a browser extension for Chrome, Edge, Firefox, Brave, and Safari. It converts Gregorian dates on web pages to Jalali/Shamsi or Lunar Hijri dates and supports per-site controls for numbers, fonts, RTL/LTR direction, text alignment, date format, and time format.

> "Calendars differ; time should remain readable for everyone."
>
> MHitMaN

## Features
- Gregorian to Jalali or Lunar Hijri conversion for common Date, ISO, RFC, numeric, and textual formats.
- Relative time conversion and Shadow DOM text processing.
- Persian, Arabic, and English number conversion.
- AM/PM localization and optional time conversion by selected time zone.
- Per-site settings for font, font size, direction, alignment, output language, date format, and time format.
- Datepicker, input, code block, and icon-font protection.
- SPA/AJAX and delayed content support.

## Fonts
To use a custom font, install it on your operating system. After installation, open the extension font setting and choose "Load system fonts" so installed fonts appear in the font selector.

If a browser does not expose a system-font listing API, only generic system choices such as `System UI`, `serif`, `sans-serif`, and `monospace` are shown.
